# Arona System (아로나 시스템)

**"선생님, 업무를 시작할까요?"**

블루아카이브의 '아로나'가 되어 선생님(사용자)의 서버를 관리해주는 **개인용 통합 관리 시스템**입니다.
현재는 마인크래프트 서버 모니터링이 주 기능이며, 추후 다양한 기능을 확장할 수 있습니다.

## ✨ 기능
- **서버 상태 확인** (`/상태`): 로컬에서 실행 중인 `server.jar` 프로세스를 감지합니다.
- **점검 공지** (`/점검시작`, `/점검종료`): 시작/종료 시간을 선택하여 채널에 공지합니다.
- **자동 알리미**: 서버가 켜지거나 꺼지면 CPU, 메모리 정보가 포함된 상세 리포트 카드를 자동으로 보냅니다.
- **채팅 청소** (`/채팅정리`): 채널의 메시지를 대량으로 삭제합니다.

## 🛠️ 설치 및 실행 방법

### 1. 필수 프로그램 설치
- Python 3.8 이상

### 2. 패키지 설치
```bash
pip install -r requirements.txt
```

### 3. 설정 (.env)
`.env` 파일을 생성하고 다음 내용을 입력하세요.
```env
DISCORD_TOKEN=여기에_토큰_입력
MC_PROCESS_TARGET=server.jar (감지할 파일명)
```

### 4. 실행
```bash
python3 bot.py
```

## 📝 빌드 (EXE 만들기)
Visual Studio Code 터미널 등에서 아래 명령어로 실행 파일(exe)을 생성할 수 있습니다.

```bash
# Windows 권장 명령어 (모듈 호출 방식)
python -m PyInstaller --onefile --name Aronabot bot.py
```

### ⚠️ 주의사항
생성된 `dist/Aronabot.exe` 파일을 실행할 때, 반드시 **`.env` 파일이 같은 폴더에 있어야** 봇이 정상적으로 작동합니다.
(친구에게 공유할 때도 exe 파일과 .env 파일을 함께 주세요!)
